# Settings Tab i18n — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make all static text in the Settings tab respond to language switching by wiring existing TEKST keys to the section titles, descriptions, sport pills, season format pills, and "Add season" button.

**Architecture:** Two sequential tasks. Task 1 adds `id` attributes to 9 static HTML elements in `app.html` that currently have no hook for JS to target. Task 2 updates `renderSettings()` in `settings-render.js` to set those elements via `t()`, and replaces three hardcoded inline strings in the dynamic pill rendering (sport labels, season format labels, "snart") with `t()` calls. All required TEKST keys already exist — no new keys needed.

**Tech Stack:** Vanilla ES modules — no test framework. Verification is manual in-browser by switching language and confirming the Settings tab re-renders in English.

### Task 2: Wire `t()` calls in `renderSettings()`

**Files:**
- Modify: `js/settings-render.js`

**Prerequisite:** Task 1 must be committed first.

- [ ] **Step 1: Update static section titles and descriptions**

  At the top of `renderSettings()`, after the existing Language section wiring (lines 23–26), add wiring for the remaining 8 elements. Insert immediately after line 26 (`if (el2) el2.textContent = t('lang_desc');`):

  ```javascript
  var stSportTitle = document.getElementById('st-sport-title');
  if (stSportTitle) stSportTitle.textContent = t('sport_title');
  var stSportDesc = document.getElementById('st-sport-desc');
  if (stSportDesc) stSportDesc.textContent = t('sport_desc');
  var stSfTitle = document.getElementById('st-sf-title');
  if (stSfTitle) stSfTitle.textContent = t('sf_title');
  var stSfDesc = document.getElementById('st-sf-desc');
  if (stSfDesc) stSfDesc.textContent = t('sf_desc');
  var stDfTitle = document.getElementById('st-df-title');
  if (stDfTitle) stDfTitle.textContent = t('df_title');
  var stDfDesc = document.getElementById('st-df-desc');
  if (stDfDesc) stDfDesc.textContent = t('df_desc');
  var stAsTitle = document.getElementById('st-as-title');
  if (stAsTitle) stAsTitle.textContent = t('as_title');
  var stAsDesc = document.getElementById('st-as-desc');
  if (stAsDesc) stAsDesc.textContent = t('as_desc');
  var addSeasonBtn = document.getElementById('settings-add-season-btn');
  if (addSeasonBtn) addSeasonBtn.textContent = '+ ' + t('add_item');
  ```

- [ ] **Step 2: Replace hardcoded sport pill labels with `t()`**

  Current sport pills array (line 43):
  ```javascript
  [{ key: 'fotball', label: '\u26BD Fotball', soon: false }, { key: 'orientering', label: '\u{1F9ED} Orientering', soon: true }, { key: 'ski', label: '\u26F7\uFE0F Ski', soon: true }]
  ```

  Replace with:
  ```javascript
  [{ key: 'fotball', label: t('sport_fotball'), soon: false }, { key: 'orientering', label: t('sport_ori'), soon: true }, { key: 'ski', label: t('sport_ski'), soon: true }]
  ```

  > `sport_fotball`, `sport_ori`, `sport_ski` exist in both `no` and `en` branches of TEKST and include the emoji (e.g. `'⚽ Fotball'` / `'⚽ Football'`).

- [ ] **Step 3: Replace hardcoded `'(snart)'` with `t('snart')`**


  Replace with:
  ```javascript
  btn.innerHTML = sp.label + (sp.soon ? ' <span style="font-size:10px">(' + t('snart') + ')</span>' : '');
  ```

- [ ] **Step 4: Replace hardcoded season format pill labels with `t()`**

  Current season format pills array (line 58):

  ```javascript
  [{ key: 'aar', label: '\u{1F4C5} \u00c5r (2025)' }, { key: 'sesong', label: '\u{1F5D3}\uFE0F Sesong (2025\u20132026)' }]
  ```
